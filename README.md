# blind_auction
Programa de Subasta Secreta
¡Bienvenido al Programa de Subasta Secreta! Este script en Python permite realizar una subasta secreta donde múltiples postores pueden participar y hacer sus ofertas. Al final de la subasta, el script determina y anuncia al ganador con la oferta más alta.

Características
-Recopila ofertas de múltiples participantes.
-Garantiza que las ofertas sean secretas hasta que la subasta termine.
-Determina automáticamente al ganador con la oferta más alta.
-Interfaz amigable con instrucciones y mensajes claros.

Cómo Funciona
-Inicializar Subasta: El script comienza mostrando un mensaje de bienvenida y el logo de la subasta.
-Recopilar Ofertas: Solicita a cada participante que ingrese su nombre y monto de oferta.
-Verificar Más Postores: Después de cada oferta, el script pregunta si hay más participantes.
-Determinar Ganador: Una vez recopiladas todas las ofertas, el script calcula la oferta más alta y anuncia al ganador.
